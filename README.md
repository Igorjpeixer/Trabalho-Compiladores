# Trabalho de Compiladores
Alunos: Igor José Peixer e Rodrigo Spicker

## Instruções

Os comandos da linguagem são em **português**, mas a **declaração de variáveis no estilo C**.
Tipos: `int`, `float`, `bool`, `char`, `string` e `void`.
Comandos de controle: `se`, `senao`, `enquanto`, `para` e `faca`.
O retorno de função: `retorne`.
Entrada e saída: `leia` e `escreva`

## Como compilar e executar o compilador

1. Abra o arquivo de projeto `.pro` no Qt Creator.
2. Compile com Build e rode com Run.
